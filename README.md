# 🎯 SyndicateNexus Tracker

> Suivi visuel du projet **SyndicateNexus** — 21 jours de build in public pour créer un Operating System à Agents IA.

## 🚀 Lien en production

🔗 **[syndicate-nexus-tracker.vercel.app](https://syndicate-nexus-tracker.vercel.app)**

## 📋 Contenu

- **Hero** : présentation du projet
- **Stats** : chiffres clés
- **Agents** : la crew d'agents IA
- **Architecture** : arbre des microservices
- **Roadmap** : timeline des 3 semaines
- **Kanban** : suivi des tâches
- **LinkedIn** : calendrier de publication

## 🛠️ Tech

- Next.js 16
- TypeScript
- Tailwind CSS
- Framer Motion
- Recharts
- Vercel

## 📦 Développement

```bash
npm install
npm run dev
```

## 🌐 Déploiement

Le site est déployé automatiquement sur **Vercel** à chaque push sur `main`.
